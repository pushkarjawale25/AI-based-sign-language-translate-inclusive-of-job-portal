# AI-based-sign-language-translate-inclusive-of-job-portal

An intelligent, real-time assistive web platform that converts Sign Language to Text, Text to Sign, supports Grammar Correction, Text-to-Speech, and provides an Inclusive Job Portal for hearing and speech-impaired individuals.

Project Overview

Communication barriers faced by hearing and speech-impaired individuals often limit social interaction and employment opportunities. This project bridges that gap by using Computer Vision, Machine Learning, and Web Technologies to enable seamless two-way communication and inclusive job access.

Key Features

Real-Time Sign Language to Text Conversion

Hand Landmark Detection using MediaPipe

Machine Learning-based Gesture Classification

Automatic Grammar Correction

Text-to-Speech Output

Text to Sign Language Visualization

Inclusive Job Portal for Differently-Abled Users

Web-Based Interface using Flask
